# Hello Cargo

Este es un repositorio simple creado para probar y aprender el lenguaje de programación **Rust**.

## Descripción

El propósito de este proyecto es experimentar con las funcionalidades básicas de Rust y el gestor de paquetes Cargo.

## Cómo ejecutar

Para ejecutar este proyecto, asegúrate de tener Rust y Cargo instalados. Luego, corre el siguiente comando en tu terminal:

```bash
cargo run
```

## Estructura

- `src/main.rs`: Contiene el código fuente principal.
- `Cargo.toml`: Archivo de configuración del proyecto y dependencias.
